# Release Notes -- June 2026

## Brand voice checks
The content-marketing brand-voice checker is now available. It validates
marketing copy against a codified brand voice on every pull request and reports
each violation inline with the rule that fired.

## Channel variants
Social post generation now produces per-channel variants for LinkedIn, X,
Facebook, Instagram, and Threads. Each channel respects its own length limit.

## Cost controls
The preview endpoint now estimates cost per channel and blocks any run that
exceeds the configured budget.
