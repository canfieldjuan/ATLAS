# Import-debt allowlist for the extracted_competitive_intelligence scaffold.
#
# scripts/check_extracted_competitive_intelligence_imports.py walks every
# Python file in the scaffold's manifest and audits its relative
# imports. Any unresolved import that does not appear in this file
# fails the check.
#
# As of the current commit, the resolver honors Python's relative-
# import semantics (ascending ``level - 1`` package components) and
# every relative import in the scaffolded modules either:
#
#   - resolves inside the scaffold itself, or
#   - resolves via the atlas_brain fallback path because the scaffold
#     has not yet copied that module.
#
# This file is therefore empty by design. Future Phase 2/3 work that
# builds out a standalone substrate or rewires modules via Protocol-
# based DI seams may temporarily add intentional debt entries here;
# each should reference its tracking issue.
#
# Format: one entry per line. Lines starting with '#' are comments.
